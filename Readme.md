## 🎊 Getting Started

```
  $ git fork
  $ git clone
  $ yarn install
  $ yarn start
```

## 🎠 About

- 내 등산 플로깅, **원스톱으로는 불가능한걸까...?** 😥
- **걱정말아요, `PLOVO`가 있잖아요!**
  1. 지금 플로깅하기 좋은 산을 실시간으로 확인하세요.
  2. 플로깅 시작 전에는 `Start`버튼을 꾸욱-⭐
  3. 플로깅 기록을 예쁜 카드로 만들어서 인스타그램에 공유할 수 있어요.
  4. 나의 4주치 기록을 한 눈에 확인해봐요!
  5. 남들은 어떤 산을 플로깅을 하는지, 볼 수 있어요.
